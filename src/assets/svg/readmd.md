# svg 图标

彩色字体库 ！！！！！！！ 使用请看说明

使用规则，复制iconfont生成的svg 文件后，需要做如下处理

1. 替换所有的彩色填充 #FDF187 改为 currentColor
2. 颜色接近 #FDF187 的统一使用 #FDF187
3. 使用svg的color默认限时色为灰色，激活色为 #FDF187
4. 如需特殊处理，请说明
